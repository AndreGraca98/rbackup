# Rust (how to)

## USER

### Install

`cargo install --git https://github.com/AndreGraca98/rbackup.git `

## DEV

### Create executables

1. Debug

   `cargo build`
2. Release

   `cargo build -r`

### Running executables

1. Debug

   `targets/debug/<myapp>`
2. Release

   `targets/release/<myapp>`
